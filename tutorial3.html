Membuat Halaman Github

Beberapa cara yang bisa Anda gunakan untuk membuat halaman github :

1.) Buat Repositori Baru

Buatlah repositori baru dengan nama username.github.io. Gunakan username github Anda

2.) Buat Repositri di Komputer

Buka terminal atau CMD, kemudian buatlah repositori baru. Kita akan mengisi, repositori ini dengan file index.html

3.) Tambahkan URL Remote Repository

Silahkan ketik perintah berikut, sesuaikan dengan URL repository anda.

4.) Kirim ke Github

Sinkronkan repositori lokal dengan repositori remote di Github.

5.) Coba Lihat Hasilnya

Coba buka alamat github yang anda buat, selanjutnya anda bisa kembangkan sendiri websitenya.
